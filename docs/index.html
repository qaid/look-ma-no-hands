🙌🏾 Look Ma,
No Hands

Speak anywhere on your Mac. Whisper transcribes locally in under a second. No cloud. No accounts. Your voice stays yours.

Features
A1 Voice Dictation Press your hotkey, speak, and formatted text appears at your cursor in any app. ~1 second with WhisperKit on Apple Silicon. ~1s
A2 Meeting Capture Record system audio from Zoom, Meet, or Teams. Continuous real-time transcription with live display. Live
B1 AI Meeting Notes Transform raw transcripts into structured notes with decisions, action items, and summaries. Ollama runs locally. Local
B2 Private by Design Zero cloud processing. Zero telemetry. Zero accounts. Works fully offline after initial model download. 0 req

How It Works
1

Install

One Homebrew command, or download the signed and notarized app from GitHub Releases.

2

Press Caps Lock

Or set any custom hotkey. A floating indicator appears to confirm you're recording.

3

Speak

Talk naturally. Press the key again. Formatted text is inserted where your cursor sits.

Install

$ brew tap qaid/lmnh $ brew install --cask look-ma-no-hands

Or download the latest .dmg from GitHub Releases

Requirements

  • PlatformmacOS 14+ (Sonoma)
  • ArchitectureApple Silicon recommended
  • Disk Space~200 MB for Whisper model
  • AI NotesOllama (optional)
